#**数字类型的比较查询**
#####例如需要查询用户表中年龄大于25的所有人的数据信息
    mongodbDao.query('User',{age:{$gt:25}}).then(function(data){
    })
#####其中的$gt的意思就是“大于”，另外mongodb还提供了很多比较符
|    符号    |    含义   |   
| :--------:| :--------: | 
|   $lt     |    <      |  
|   $lte   |    <=    |
|   $gt    |    >     |
|   $gte   |     >=   |
|   $ne    |     !=   |
